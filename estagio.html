Relatório de Estágio

INTRODUÇÂO

O estágio foi realizado na empresa SEMEAR, especializada em consultoria para fazendas e propriedades rurais, utilizando drones para análise de terras e monitoramento de cultivos. O principal objetivo foi aplicar tecnologias inovadoras na agricultura de precisão.

1.1. Objetivos

O objetivo geral deste estágio foi contribuir para o desenvolvimento de soluções tecnológicas no monitoramento e análise de propriedades rurais.

1.1.2. Objetivos Específicos

Auxiliar na coleta de dados geoespaciais com drones. Interpretar imagens de satélite e de drones para fornecer relatórios detalhados aos agricultores. Colaborar no desenvolvimento de sistemas de monitoramento de terras para detectar áreas com risco de erosão ou pragas.

Atividades realizadas

2.1. Captura de Imagens com Drones

Durante o estágio, participei da operação de drones para capturar imagens de alta resolução das fazendas.

2.2. Interpretação de Imagens

Após a coleta dos dados, ajudei a interpretar as imagens capturadas para gerar relatórios precisos sobre as condições das terras.

Conclusão

Este estágio foi uma experiência enriquecedora, permitindo a aplicação de tecnologias avançadas para resolver problemas reais enfrentados por produtores rurais. A utilização de drones para monitoramento de terras trouxe um novo nível de eficiência.
